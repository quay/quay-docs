= getOrgImmutabilityPolicy

[role="_abstract"]
Get Org Immutability Policy.

[id="getorgimmutabilitypolicy-get"]
== GET /api/v1/organization/{orgname}/immutabilitypolicy/{policy_uuid}



**Authorizations: **oauth2_implicit (**org:admin**)
