= Enabling features after deployment

[role="_abstract"]
To enable new features for your {productname} registry after deployment, you can edit the `configBundleSecret` resource. You can use the {ocp} web console or the CLI to make these changes.

[NOTE]
====
Using the {ocp} web console to enable features is generally considered a simpler method.
====
