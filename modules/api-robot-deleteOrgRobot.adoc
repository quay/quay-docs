[id="deleteorgrobot"]
= deleteOrgRobot

[role="_abstract"]
Delete an existing organization robot.

[id="deleteorgrobot-delete"]
== DELETE /api/v1/organization/{orgname}/robots/{robot_shortname}



**Authorizations: **oauth2_implicit (**org:admin**)


[id="deleteorgrobot-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**robot_shortname** + 
_required_|The short name for the robot, without any user or organization prefix|string
|path|**orgname** + 
_required_|The name of the organization|string
|===


[id="deleteorgrobot-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|204|Deleted|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="deleteorgrobot-example-command"]
== Example command

[source,terminal]
----
curl -X DELETE \
  -H "Authorization: Bearer <bearer_token>" \
  "<quay-server.example.com>/api/v1/organization/<organization_name>/robots/<robot_shortname>"
----
