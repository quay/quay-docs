= getOrganization
Get the details for the specified organization.

[discrete]
== GET /api/v1/organization/{orgname}



**Authorizations: **
[discrete]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**orgname** + 
_required_|The name of the organization|string
|===


[discrete]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

== Response body schema (application/json)

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**name**|Organization name|string
|**email**|For backward compatibility, returns the value of `contact_email` (or empty string if null). Visible only to org admins and superusers.|string
|**contact_email**|Email address for receiving automated system alerts (quota warnings, security notifications). Can be shared across multiple organizations. Returns `null` if not configured (notifications default to org owners). Visible only to org admins and superusers.|string or null
|**avatar**|Avatar information for the organization.|object
|**is_admin**|Determines if the current user is an admin of the organization.|boolean
|**is_member**|Determines if the current user is a member of the organization.|boolean
|**teams**|If the user is a member of the organization, states the team within the organization.|object
|**ordered_teams**|Alphabetically ordered list of team names.|array of strings
|**invoice_email**|Determines if an invoice email is set.|boolean
|**invoice_email_address**|If set, specifies the invoice email address.|string or null
|**tag_expiration_s**|Tag expiration time in seconds.|integer
|**is_free_account**|Specifies a free account.|boolean
|===
