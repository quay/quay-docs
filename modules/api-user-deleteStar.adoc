[id="deletestar"]
= deleteStar

[role="_abstract"]
Removes a star from a repository.

[id="deletestar-delete"]
== DELETE /api/v1/user/starred/{repository}



**Authorizations: **oauth2_implicit (**user:admin**)


[id="deletestar-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**repository**
 
_required_|The full path of the repository. e.g. namespace/name|string
|===


[id="deletestar-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|204|Deleted|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="deletestar-example-command"]
== Example command

[source,terminal]
----
$ curl -X DELETE "https://quay-server.example.com/api/v1/user/starred/namespace/repository-name" \
  -H "Authorization: Bearer <your_access_token>"
----
