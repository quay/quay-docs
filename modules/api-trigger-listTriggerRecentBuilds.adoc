[id="listtriggerrecentbuilds"]
= listTriggerRecentBuilds

[role="_abstract"]
List the builds started by the specified trigger.

[id="listtriggerrecentbuilds-get"]
== GET /api/v1/repository/{repository}/trigger/{trigger_uuid}/builds



**Authorizations: **oauth2_implicit (**repo:admin**)


[id="listtriggerrecentbuilds-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**trigger_uuid**
 
_required_|The UUID of the build trigger|string
|path|**repository**
 
_required_|The full path of the repository. e.g. namespace/name|string
|===


[id="listtriggerrecentbuilds-query-parameters"]
== Query parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|query|**limit**
 
_optional_|The maximum number of builds to return|integer
|===


[id="listtriggerrecentbuilds-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema

|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;

|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="listtriggerrecentbuilds-example-command"]
== Example command

----
