= Deploy the {productname} registry by using the CLI

[role="_abstract"]
To deploy your proof of concept {productname} registry, you can start the registry container with Podman.

.Procedure

* Enter the following command to start the `Quay` registry container, specifying the appropriate volumes for configuration data and local storage for image data:
+
[subs="verbatim,attributes"]
----
$ sudo podman run -d --rm -p 80:8080 -p 443:8443  \
   --name=quay \
   -v $QUAY/config:/conf/stack:Z \
   -v $QUAY/storage:/datastorage:Z \
   {productrepo}/{quayimage}:{productminv}
----
