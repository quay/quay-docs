[id="getuserautoprunepolicy"]
= getUserAutoPrunePolicy

[role="_abstract"]
Fetches the auto-prune policy for the currently logged in user

[id="getuserautoprunepolicy-get"]
== GET /api/v1/user/autoprunepolicy/{policy_uuid}



**Authorizations: **oauth2_implicit (**user:admin**)


[id="getuserautoprunepolicy-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**policy_uuid**
 
_required_|The unique ID of the policy|string
|===


[id="getuserautoprunepolicy-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="getuserautoprunepolicy-example-command"]
== Example command

$ curl -X GET "https://quay-server.example.com/api/v1/user/autoprunepolicy/{policy_uuid}" \
