= CVE ratings from the National Vulnerability Database

[role="_abstract"]
Clair enriches vulnerability findings with Common Vulnerability Scoring System (CVSS) scores from the National Vulnerability Database. You can view CVSS enrichment data in the {productname} UI.

With this change, if the vulnerability has a CVSS score that is within 2 levels of the distribution score, the {productname} UI presents the distribution's score by default. For example:

image:clair-4-2-enrichment-data.png[Clair v4.2 data display]

This differs from the previous interface, which would only display the following information:

image:clair-4-0-cve-report.png[Clair v4 data display]
