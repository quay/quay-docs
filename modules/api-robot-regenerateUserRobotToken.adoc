[id="regenerateuserrobottoken"]
= regenerateUserRobotToken

[role="_abstract"]
Regenerates the token for a user's robot.

[id="regenerateuserrobottoken-post"]
== POST /api/v1/user/robots/{robot_shortname}/regenerate



**Authorizations: **oauth2_implicit (**user:admin**)


[id="regenerateuserrobottoken-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**robot_shortname** + 
_required_|The short name for the robot, without any user or organization prefix|string
|===


[id="regenerateuserrobottoken-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|201|Successful creation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="regenerateuserrobottoken-example-command"]
== Example command
[source,terminal]
----
$ curl -X POST \
  -H "Authorization: Bearer <bearer_token>" \
  "<quay-server.example.com>/api/v1/user/robots/<robot_shortname>/regenerate"
----
