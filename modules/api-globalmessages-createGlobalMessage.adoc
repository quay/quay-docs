[id="createglobalmessage"]
= createGlobalMessage

[role="_abstract"]
Create a message.

[id="createglobalmessage-post"]
== POST /api/v1/messages



**Authorizations: **oauth2_implicit (**super:user**)



[id="createglobalmessage-request-body-schema"]
== Request body schema (application/json)

Create a new message

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**message**
 
_required_|A single message|object
|===


[id="createglobalmessage-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|201|Successful creation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="createglobalmessage-example-command"]
== Example command
[source,terminal]
----
$ curl -X POST "https://<quay-server.example.com>/api/v1/messages" \
    -H "Authorization: Bearer <access_token>" \
    -H "Content-Type: application/json" \
    -d '{
        "message": {
            "content": "Hi",
            "media_type": "text/plain",
            "severity": "info"
        }
    }'
----
