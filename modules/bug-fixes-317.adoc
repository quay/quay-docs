= {productname} bug fixes

The following issues were fixed with {productname} 3.17:

* link:https://issues.redhat.com/browse/PROJQUAY-9668[*PROJQUAY-9668*]. In this update, numerous LDAP queries during login due to various organization memberships previously resulted in timeouts and prevented user login. In this release, we've decreased the excessive LDAP queries during login. Consequently, LDAP user login no longer times out due to excessive queries, enhancing login speed for affected users.

* link:https://issues.redhat.com/browse/PROJQUAY-9720[*PROJQUAY-9720*]. In this update, the upstream build process now actively excludes the `.npm` directory from the image, as a `.dockerignore` for it was previously missing. Consequently, the NPM cache no longer persists in local development images, reducing image size. With this release, the NPM cache is no longer incorporated in local Quay image builds. As a result, the NPM cache will not be present in locally built upstream images, ensuring faster and cleaner builds.
