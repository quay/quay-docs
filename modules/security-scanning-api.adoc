= View Clair security scans by using the API

You can view Clair security scans by using the API.

.Procedure 

* Use the link:https://docs.redhat.com/en/documentation/red_hat_quay/{producty}/html-single/red_hat_quay_api_guide/index#getrepomanifestsecurity[`GET /api/v1/repository/{repository}/manifest/{manifestref}/security`] endpoint to retrieve security information about a specific manifest in a repository. For example:
+
[source,terminal]
----
$ curl -X GET \
  -H "Authorization: Bearer <bearer_token>" \
  -H "Accept: application/json" \
  "https://quay-server.example.com/api/v1/repository/<namespace>/<repository>/manifest/<manifest_digest>/security?vulnerabilities=<true_or_false>"
----
+
.Example output
+
[source,terminal]
----
{"status": "queued", "data": null}
----
