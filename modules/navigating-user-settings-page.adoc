= Navigating to the User Settings page 

[role="_abstract"]
To open the User Settings page in {quayio}, you can use the account menu in the header. You select *Account Settings* to open the page.

.Procedure 

. On {quayio}, click your username in the header.

. Select *Account Settings*. You are redirected to the *User Settings* page.
