= createUserQuotaSuperUser

[role="_abstract"]
Create User Quota Super User.

[id="createuserquotasuperuser-post"]
== POST /api/v1/superuser/organization/{namespace}/quota



**Authorizations: **oauth2_implicit (**super:user**)
