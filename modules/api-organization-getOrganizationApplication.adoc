[id="getorganizationapplication"]
= getOrganizationApplication

[role="_abstract"]
Retrieves the application with the specified client_id under the specified organization.

[id="getorganizationapplication-get"]
== GET /api/v1/organization/{orgname}/applications/{client_id}



**Authorizations: **oauth2_implicit (**org:admin**)


[id="getorganizationapplication-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**client_id** + 
_required_|The OAuth client ID|string
|path|**orgname** + 
_required_|The name of the organization|string
|===


[id="getorganizationapplication-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="getorganizationapplication-example-command"]
== Example command
[source,terminal]
----
$ curl -X GET "https://<quay-server.example.com>/api/v1/organization/<orgname>/applications/<client_id>" \
  -H "Authorization: Bearer <access_token>"
----
