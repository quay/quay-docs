[id="getrepositoryautoprunepolicy"]
= getRepositoryAutoPrunePolicy

[role="_abstract"]
Fetches the auto-prune policy for the repository

[id="getrepositoryautoprunepolicy-get"]
== GET /api/v1/repository/{repository}/autoprunepolicy/{policy_uuid}



**Authorizations: **oauth2_implicit (**repo:admin**)


[id="getrepositoryautoprunepolicy-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**policy_uuid**
 
_required_|The unique ID of the policy|string
|path|**repository**
 
_required_|The full path of the repository. e.g. namespace/name|string
|===


[id="getrepositoryautoprunepolicy-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===
