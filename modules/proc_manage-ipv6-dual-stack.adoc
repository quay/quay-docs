= IPv6 and dual-stack deployments

[role="_abstract"]
You can deploy standalone {productname} on IPv6-only or dual-stack (IPv4 and IPv6) networks. Set `FEATURE_LISTEN_IP_VERSION` in your `config.yaml` file to enable the protocol family that your environment supports.

Some storage backends have known limitations on IPv6-only networks.
