[id="getorgmirrorconfig"]
= getOrgMirrorConfig

[role="_abstract"]
Get the organization-level mirror configuration.

[id="getorgmirrorconfig-get"]
== GET /api/v1/organization/{orgname}/mirror



**Authorizations: **oauth2_implicit (**org:admin**)



[id="getorgmirrorconfig-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="getorgmirrorconfig-example-command"]
== Example command
[source,terminal]
----
$ curl -X GET \
  -H "Authorization: Bearer <bearer_token>" \
  -H "Accept: application/json" \
  https://<quay-server.example.com>/api/v1/organization/<orgname>/mirror
----
