[id="createstar"]
= createStar

[role="_abstract"]
Star a repository.

[id="createstar-post"]
== POST /api/v1/user/starred



**Authorizations: **oauth2_implicit (**repo:read**)



[id="createstar-request-body-schema"]
== Request body schema (application/json)



[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**namespace**
 
_required_|Namespace in which the repository belongs|string
|**repository**
 
_required_|Repository name|string
|===


[id="createstar-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|201|Successful creation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="createstar-example-command"]
== Example command

[source,terminal]
----
$ curl -X POST "https://quay-server.example.com/api/v1/user/starred" \
  -H "Authorization: Bearer <your_access_token>" \
  -H "Content-Type: application/json" \
  -d '{
        "namespace": "<namespace>",
        "repository": "<repository_name>"
      }'
----
