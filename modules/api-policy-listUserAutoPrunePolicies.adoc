[id="listuserautoprunepolicies"]
= listUserAutoPrunePolicies

[role="_abstract"]
Lists the auto-prune policies for the currently logged in user

[id="listuserautoprunepolicies-get"]
== GET /api/v1/user/autoprunepolicy/



**Authorizations: **oauth2_implicit (**user:admin**)



[id="listuserautoprunepolicies-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="listuserautoprunepolicies-example-command"]
== Example command

[source,terminal]
----
$ curl -X GET "https://quay-server.example.com/api/v1/user/autoprunepolicy/" \
  -H "Authorization: Bearer <your_access_token>"
