= Troubleshooting the QuayRegistry CR

[role="_abstract"]
To troubleshoot issues during {productname} registry deployment, you can check the `QuayRegistry` custom resource status using the {ocp} web console *Events* page or the `oc` CLI. These methods help you identify and resolve deployment problems.
