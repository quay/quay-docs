= Event notifications for mirroring

[role="_abstract"]
Repository mirroring in {productname} can send notifications when a mirror starts, succeeds, or fails. You can configure these events in the repository Settings tab.

Repository mirroring supports the following notification events:

* Repository Mirror Started
* Repository Mirror Success
* Repository Mirror Unsuccessful

The events can be configured inside the *Settings* tab for each repository. Existing notification methods such as email, Slack, Quay UI, and webhooks are supported.
