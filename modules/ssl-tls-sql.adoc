= Certificate-based authentication between {productname} and SQL

[role="_abstract"]
You can configure certificate-based authentication between {productname} and SQL databases such as PostgreSQL and GCP CloudSQL by supplying client-side SSL/TLS certificates. This approach verifies the server certificate against a trusted Certificate Authority and supports automated deployments.
