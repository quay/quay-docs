[id="unmanaged-clair-configuration"]
= Unmanaged Clair configuration

[role="_abstract"]
Unmanaged Clair configuration lets you run a custom Clair deployment or use an external Clair database with the {productname} Operator. You can use this option for geo-replicated environments or highly available databases outside your cluster.
