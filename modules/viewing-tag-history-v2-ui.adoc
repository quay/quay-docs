= Viewing {productname} tag history by using the UI

[role="_abstract"]
To review changes made to an image over time, you can open its *Tag History* page in the {productname} v2 UI. You can search by tag name, filter by date range, and see when each tag was modified.

.Procedure

. On the {productname} v2 UI dashboard, click *Repositories* in the navigation pane. 

. Click the name of a repository that has image tags.

. Click *Tag History*. On this page, you can perform the following actions:
+
* Search by tag name
* Select a date range 
* View tag changes
* View tag modification dates and the time at which they were changed
