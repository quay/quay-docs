= Deploying {productname} on public cloud

[role="_abstract"]
{productname} runs on public clouds in standalone mode or when {ocp} runs on public cloud. You can use public cloud backend services for high availability and scalability and consult tested integration documentation for supported configurations.

**Recommendation:** If {productname} is running on public cloud, then you should use the public cloud services for {productname} backend services to ensure proper high availability and scalability.
