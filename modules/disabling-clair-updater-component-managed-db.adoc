= Disabling the Clair Updater component

[role="_abstract"]
You can disable the Clair updater component when you run {productname} in a disconnected environment. Set `matcher.disable_updaters` to `true` in the Clair configuration.

In the following example, Clair updaters are disabled:

[source,yaml]
----
#...
matcher:
  disable_updaters: true
#...
----
