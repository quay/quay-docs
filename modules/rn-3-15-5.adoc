== {productname} 3.15.5 bug fixes

* link:https://issues.redhat.com/browse/PROJQUAY-10609[*PROJQUAY-10609*]. Previously, after the RHEL 9 image migration, image pulls could fail with a `cryptography.exceptions.UnsupportedAlgorithm: sha1 is not supported by this backend for RSA signing` error when {productname} generated presigned URLs for object storage backends such as Amazon CloudFront. This issue occurred because the deprecated RSA-SHA1 signing algorithm was blocked by the system cryptographic policy on RHEL 9.
+
With this release, {productname} no longer relies on RSA-SHA1 for presigned URL generation, and image pulls succeed without this signing error.
