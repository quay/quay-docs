= Enabling mirroring for a repository

[role="_abstract"]
To automatically synchronize container images between registries, enable mirroring for your {productname} repository. Configuring this automated synchronization ensures that your deployment always has access to the most up-to-date container files.

By default for {productname}, FEATURE_REPO_MIRROR` is set to `False`. To enable mirroring, you must set `FEATURE_REPO_MIRROR: true` in your `config.yaml` file.

.Procedure

. Set `FEATURE_REPO_MIRROR: true` in your `config.yaml` file. For example:
+
[source,yaml]
----
# ...
FEATURE_REPO_MIRROR: true
# ...
----

. Restart your {productname} registry.
