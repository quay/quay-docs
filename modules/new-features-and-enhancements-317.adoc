= Quay container image size optimization

The {productname} container image now uses Red Hat Universal Base Image (UBI) minimal 9.7 as the base layer, with build-time dependencies and caches removed from the final image. This change reduces image size by approximately 30%, shortens download and deployment times, and narrows the attack surface. The image remains a drop-in replacement for previous versions, and no configuration changes are required for upgrades to {productmin}.
