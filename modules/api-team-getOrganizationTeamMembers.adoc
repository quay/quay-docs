[id="getorganizationteammembers"]
= getOrganizationTeamMembers

[role="_abstract"]
Retrieve the list of members for the specified team.

[id="getorganizationteammembers-get"]
== GET /api/v1/organization/{orgname}/team/{teamname}/members



**Authorizations: **oauth2_implicit (**org:admin**)


[id="getorganizationteammembers-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**teamname** + 
_required_|The name of the team|string
|path|**orgname** + 
_required_|The name of the organization|string
|===


[id="getorganizationteammembers-query-parameters"]
== Query parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|query|**includePending** + 
_optional_|Whether to include pending members|boolean
|===
