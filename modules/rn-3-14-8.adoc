== {productname} 3.14.8 bug fixes

* link:https://issues.redhat.com/browse/PROJQUAY-11216[*PROJQUAY-11216*]. Previously, repository mirroring failed in {productname} 3.14.7 when you configured a Skopeo timeout interval and clicked *Enable mirror*. The mirroring API passed the `skopeo_timeout_interval` parameter, but the underlying model did not recognize it, which returned a `500` error.
+
With this release, {productname} recognizes the Skopeo timeout interval parameter when you enable repository mirroring. As a result, you can enable mirroring without this error.
