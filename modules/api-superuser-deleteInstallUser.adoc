[id="deleteinstalluser"]
= deleteInstallUser

[role="_abstract"]
Deletes a user.

[id="deleteinstalluser-delete"]
== DELETE /api/v1/superuser/users/{username}



**Authorizations: **oauth2_implicit (**super:user**)



[id="deleteinstalluser-request-body-schema"]
== Request body schema (application/json)

Data for deleting a user

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**username** + 
_required_|The username of the user being deleted|string
|===


[id="deleteinstalluser-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|201|Successful creation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="deleteinstalluser-example-command"]
== Example command

[source,terminal]
----
$ curl -X DELETE -H "Authorization: Bearer <bearer_token>" "https://<quay-server.example.com>/api/v1/superuser/users/{username}"
