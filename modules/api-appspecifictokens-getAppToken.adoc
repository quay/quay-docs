[id="getapptoken"]
= getAppToken

[role="_abstract"]
Returns a specific app token for the user.

[id="getapptoken-get"]
== GET /api/v1/user/apptoken/{token_uuid}



**Authorizations: **oauth2_implicit (**user:admin**)


[id="getapptoken-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**token_uuid** + 
_required_|The uuid of the app specific token|string
|===


[id="getapptoken-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="getapptoken-example-command"]
== Example command
[source,terminal]
----
$ curl -X GET \
  -H "Authorization: Bearer <access_token>" \
  "http://quay-server.example.com/api/v1/user/apptoken/<token_uuid>"
----
