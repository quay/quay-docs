= Geo-replication

[role="_abstract"]
Geo-replication connects multiple geographically distributed {productname} deployments as a single registry from the client perspective. You gain faster push and pull performance in global setups along with transparent failover and client redirect.

Geo-replication is supported on standalone and Operator-based deployments.
