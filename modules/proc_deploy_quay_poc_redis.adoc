= Configuring Redis

[role="_abstract"]
To configure Redis for your proof of concept {productname} deployment, you can start a Redis container with Podman and expose port 6379 with a password. {productname} uses Redis for live builder logs during image builds.

Redis is a key-value store that is used by {productname} for live builder logs.

.Procedure

* Start the `Redis` container, specifying the port and password, by entering the following command: 
+
[subs="verbatim,attributes"]
----
$ sudo podman run -d --rm --name redis \
  -p 6379:6379 \
  -e REDIS_PASSWORD=strongpassword \
  {redisimage}
----
