= Managing auto-pruning policies using the {productname} UI

[role="_abstract"]
You can manage most {productname} auto-pruning policies from the v2 UI or the API after you enable auto-pruning and the v2 UI in your `config.yaml` file. Registry-wide policies are configured only in the `config.yaml` file.
