= Configuring the Podman CNI to use IPv6

[role="_abstract"]
You can configure the Podman Container Network Interface for IPv4 and IPv6 to run a local {productname} instance over IPv6 for development or testing. By default, the root user Podman network uses IPv4 only.

In some cases, you might want to run a local instance of {productname} to use IPv6. This setup is common for development or testing purposes.
