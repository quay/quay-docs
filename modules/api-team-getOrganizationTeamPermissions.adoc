[id="getorganizationteampermissions"]
= getOrganizationTeamPermissions

[role="_abstract"]
Returns the list of repository permissions for the org's team.

[id="getorganizationteampermissions-get"]
== GET /api/v1/organization/{orgname}/team/{teamname}/permissions



**Authorizations: **
[id="getorganizationteampermissions-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**teamname**
 
_required_|The name of the team|string
|path|**orgname**
 
_required_|The name of the organization|string
|===


[id="getorganizationteampermissions-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===
