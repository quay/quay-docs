[id="getrepo"]
= getRepo

[role="_abstract"]
Fetch the specified repository.

[id="getrepo-get"]
== GET /api/v1/repository/{repository}



**Authorizations: **oauth2_implicit (**repo:read**)


[id="getrepo-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**repository** + 
_required_|The full path of the repository. e.g. namespace/name|string
|===


[id="getrepo-query-parameters"]
== Query parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|query|**includeTags** + 
_optional_|Whether to include repository tags|boolean
|query|**includeStats** + 
_optional_|Whether to include action statistics|boolean
|===


[id="getrepo-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="getrepo-example-command"]
== Example command

[source,terminal]
----
$ curl -X GET -H "Authorization: Bearer <bearer_token>" "<quay-server.example.com>/api/v1/repository/<namespace>/<repository_name>"
----
