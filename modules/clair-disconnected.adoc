= Clair in disconnected environments

[role="_abstract"]
Clair supports disconnected {productname} deployments that have no direct internet access. You can use the `clairctl` tool to transfer vulnerability database updates from an open host so Clair can scan images offline.

Clair uses a set of components called _updaters_ to handle the fetching and parsing of data from various vulnerability databases. By default, updaters pull vulnerability data directly from the internet and work for immediate use.

[NOTE]
====
Currently, Clair enrichment data is CVSS data. Enrichment data is currently unsupported in disconnected environments.
====
