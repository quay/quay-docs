= Checking data replication in backend buckets 

[role="_abstract"]
To ensure that your {productname} data is properly replicated in all backend buckets, you can use the `aws CLI` to list objects in the bucket. Run `aws s3 ls` with `--recursive`, `--human-readable`, and `--summarize` to verify replication.

.Prerequisites 

* You have installed the `aws` CLI. 

.Procedure

. Enter the following command to ensure that your data is replicated in all backend buckets:
+
[source,terminal]
----
$ aws --profile quay_prod_s3 --endpoint=http://10.0.x.x:port s3 ls ocp-quay --recursive --human-readable --summarize
----
+
.Example output
[source,terminal]
----
Total Objects: 17996
Total Size: 514.4 GiB
----
