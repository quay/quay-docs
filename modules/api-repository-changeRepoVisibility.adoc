[id="changerepovisibility"]
= changeRepoVisibility

[role="_abstract"]
Change the visibility of a repository.

[id="changerepovisibility-post"]
== POST /api/v1/repository/{repository}/changevisibility



**Authorizations: **oauth2_implicit (**repo:admin**)


[id="changerepovisibility-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**repository** + 
_required_|The full path of the repository. e.g. namespace/name|string
|===


[id="changerepovisibility-request-body-schema"]
== Request body schema (application/json)

Change the visibility for the repository.

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**visibility** + 
_required_|Visibility which the repository will start with|string
|===


[id="changerepovisibility-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|201|Successful creation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="changerepovisibility-example-command"]
== Example Command

[source,terminal]
----
$ curl -X POST \
  -H "Authorization: Bearer <ACCESS_TOKEN>" \
  -H "Content-Type: application/json" \
  -d '{
        "visibility": "private"
      }' \
  "https://quay-server.example.com/api/v1/repository/<NAMESPACE>/<REPO_NAME>/changevisibility"
----
