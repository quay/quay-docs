[id="createorgimmutabilitypolicy"]
= createOrgImmutabilityPolicy

[role="_abstract"]
Create Org Immutability Policy.

[id="createorgimmutabilitypolicy-post"]
== POST /api/v1/organization/{orgname}/immutabilitypolicy/



**Authorizations: **oauth2_implicit (**org:admin**)



[id="createorgimmutabilitypolicy-request-body-schema"]
== Request body schema (application/json)

The immutability policy configuration

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**tagPattern** + 
_optional_|Regex pattern to match tag names|string
|**tagPatternMatches** + 
_optional_|If true, matching tags are immutable. If false, non-matching tags are immutable.|boolean
|===


[id="createorgimmutabilitypolicy-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema

|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="createorgimmutabilitypolicy-example-command"]
== Example command

[source,terminal]
----
$ curl -X POST "https://<quay-server.example.com>/api/v1/organization/<orgname>/immutabilitypolicy/" \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{"tagPattern": ".*", "tagPatternMatches": true}'
----
