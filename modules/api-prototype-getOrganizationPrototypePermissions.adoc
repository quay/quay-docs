[id="getorganizationprototypepermissions"]
= getOrganizationPrototypePermissions

[role="_abstract"]
List the existing prototypes for this organization.

[id="getorganizationprototypepermissions-get"]
== GET /api/v1/organization/{orgname}/prototypes



**Authorizations: **oauth2_implicit (**org:admin**)


[id="getorganizationprototypepermissions-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**orgname** + 
_required_|The name of the organization|string
|===
