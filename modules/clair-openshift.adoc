= Clair on {ocp}

[role="_abstract"]
The {productname} Operator automatically installs and configures Clair when you deploy {productname} on {ocp}. This simplifies setup by eliminating the need for manual Clair configuration.

If your registry scans large images, you can increase Clair scratch storage for image layer extraction by configuring ephemeral volume overrides on the `clair` component. For more information, see "Configuring ephemeral storage for managed Clair".
