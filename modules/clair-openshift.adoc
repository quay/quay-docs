= Clair on {ocp}

To set up Clair v4 (Clair) on a {productname} deployment on {ocp}, it is recommended to use the {productname} Operator. By default, the {productname} Operator installs or upgrades a Clair deployment along with your {productname} deployment and configure Clair automatically.
