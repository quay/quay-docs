[id="deletefulltag"]
= deleteFullTag

[role="_abstract"]
Delete the specified repository tag.

[id="deletefulltag-delete"]
== DELETE /api/v1/repository/{repository}/tag/{tag}



**Authorizations: **oauth2_implicit (**repo:write**)


[id="deletefulltag-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**repository**
 
_required_|The full path of the repository. e.g. namespace/name|string
|path|**tag**
 
_required_|The name of the tag|string
|===


[id="deletefulltag-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|204|Deleted|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="deletefulltag-example-command"]
== Example command

[source,terminal]
----
$ curl -X DELETE "https://quay-server.example.com/api/v1/repository/<namespace>/<repo_name>/tag/<tag_name>" \
  -H "Authorization: Bearer <your_access_token>"
----
