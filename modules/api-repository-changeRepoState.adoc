[id="changerepostate"]
= changeRepoState

[role="_abstract"]
Change the state of a repository.

[id="changerepostate-put"]
== PUT /api/v1/repository/{repository}/changestate



**Authorizations: **oauth2_implicit (**repo:admin**)


[id="changerepostate-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**repository** + 
_required_|The full path of the repository. e.g. namespace/name|string
|===


[id="changerepostate-request-body-schema"]
== Request body schema (application/json)

Change the state of the repository.

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**state** + 
_required_|Determines whether pushes are allowed.|string
|===
