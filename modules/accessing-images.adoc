= Accessing images

[role="_abstract"]
To access {productname} and Clair images for standalone upgrades, pull from registry.redhat.io or registry.access.redhat.com and configure authentication as described in Red Hat Container Registry Authentication.

{productname} images from version 3.4.0 and later are available from registry.redhat.io and registry.access.redhat.com. Configure authentication before pulling images.

[id="upgrade-to-producty-from-3_17_z"]
== Upgrade to {productminv} from 3.17.z

* **Quay:** {productrepo}/{quayimage}:{productminv}
ifdef::downstream[]
* **Clair:** {productrepo}/{clairimage}:{productminv}
endif::downstream[]
ifdef::upstream[]
* **Clair:** {productrepo}/{clairimage}:{clairproductminv}
endif::upstream[]
* **PostgreSQL:** {postgresimage}
* **Redis:** {redisimage}
* **Clair-PostgreSQL:** {postgresimage}
