= DNS configuration for a manual cluster

[role="_abstract"]
To manage a Prometheus DNS record outside Kubernetes, you can use SkyDNS with an etcd cluster to track {productname} instance addresses.

SkyDNS can run on an etcd cluster. Entries for each {productname} instance in the cluster can be added and removed in the etcd store. SkyDNS regularly reads them from there and updates the list of {productname} instances in the DNS record accordingly.
