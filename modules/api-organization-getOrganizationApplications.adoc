[id="getorganizationapplications"]
= getOrganizationApplications

[role="_abstract"]
List the applications for the specified organization.

[id="getorganizationapplications-get"]
== GET /api/v1/organization/{orgname}/applications



**Authorizations: **oauth2_implicit (**org:admin**)


[id="getorganizationapplications-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**orgname**
 
_required_|The name of the organization|string
|===


[id="getorganizationapplications-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="getorganizationapplications-example-command"]
== Example command

[source,terminal]
----
$ curl -X GET "https://<quay-server.example.com>/api/v1/organization/<orgname>/applications" \
  -H "Authorization: Bearer <access_token>"
----
