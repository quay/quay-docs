[id="getmatchingentities"]
= getMatchingEntities

[role="_abstract"]
Get a list of entities that match the specified prefix.

[id="getmatchingentities-get"]
== GET /api/v1/entities/{prefix}



**Authorizations: **
[id="getmatchingentities-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**prefix**
 
_required_||string
|===


[id="getmatchingentities-query-parameters"]
== Query parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|query|**includeOrgs**
 
_optional_|Whether to include orgs names.|boolean
|query|**includeTeams**
 
_optional_|Whether to include team names.|boolean
|query|**namespace**
 
_optional_|Namespace to use when querying for org entities.|string
|===


[id="getmatchingentities-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="getmatchingentities-example-command"]
== Example command
[source,terminal]
----
$ curl -X GET "https://quay-server.example.com/api/v1/entities/<prefix>?includeOrgs=<true_or_false>&includeTeams=<true_or_false>&namespace=<namespace>" \
  -H "Authorization: Bearer <bearer_token>"
----
