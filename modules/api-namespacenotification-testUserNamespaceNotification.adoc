= testUserNamespaceNotification

[role="_abstract"]
Test User Namespace Notification.

[id="testusernamespacenotification-post"]
== POST /api/v1/user/namespacenotifications/{uuid}/test



**Authorizations: **oauth2_implicit (**user:admin**)


[id="testusernamespacenotification-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**uuid**
 
_required_|The uuid of the notification|string
|===


[id="testusernamespacenotification-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="testusernamespacenotification-example-command"]
== Example command

[source,terminal]
----
$ curl -X POST \
-H "Authorization: Bearer <bearer_token>" \
https://<quay-server.example.com>/api/v1/user/namespacenotifications/<uuid>/test
----
