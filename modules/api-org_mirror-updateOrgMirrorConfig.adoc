
= updateOrgMirrorConfig
Update organization mirror configuration.

[discrete]
== PUT /api/v1/organization/{orgname}/mirror



**Authorizations: **oauth2_implicit (**org:admin**)



[discrete]
== Request body schema (application/json)

Update organization mirror configuration

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**is_enabled** + 
_optional_||boolean
|**external_registry_url** + 
_optional_||string
|**external_namespace** + 
_optional_||string
|**robot_username** + 
_optional_||string
|**visibility** + 
_optional_||string
|**sync_interval** + 
_optional_||integer
|**sync_start_date** + 
_optional_||string
|**external_registry_username** + 
_optional_||
|**external_registry_password** + 
_optional_||
|**external_registry_config** + 
_optional_||object
|**repository_filters** + 
_optional_||array of string + 

|**skopeo_timeout** + 
_optional_||integer
|===

== Example command
[source,terminal]
----
$ curl -X PUT \
  -H "Authorization: Bearer <bearer_token>" \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{"is_enabled": true, "sync_interval": 7200}' \
  https://<quay-server.example.com>/api/v1/organization/<orgname>/mirror
