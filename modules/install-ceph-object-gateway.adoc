= Installing the Ceph Object Gateway

[role="_abstract"]
To expose Ceph storage to {productname}, you can install a Ceph Object Gateway on the RADOS gateway node and connect it to your Ceph cluster.

On the Ansible system (ceph05), configure a
Ceph Object Gateway to your Ceph Storage cluster (which will ultimately run on ceph04). See _Additional resources_ for details.
