= Configuring action log storage for Splunk

[role="_abstract"]
Splunk is an alternative to Elasticsearch for storing and analyzing {productname} action logs. You can forward logs directly to Splunk or to the Splunk HTTP Event Collector (HEC) during or after deployment.
