= Known issues and limitations

The following sections note known issues and limitations for {productname} 3.16.

[id="build-logs-download-error-v2-ui"]
== Cannot download build logs from the v2 UI

After an image is built successfully when using the {productname} v2 UI, users are unable to download the build logs. Attempting to click the *Download* button on the v2 UI results in the following error: `404 Not Found`.

As a temporary work around, you can download build logs by using the v1 UI.
