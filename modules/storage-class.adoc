= StorageClass

[role="_abstract"]
The {productname} Operator uses the default `StorageClass` in your cluster to provision persistent storage for the Quay and Clair PostgreSQL databases. Ensure that your cluster has a default `StorageClass` configured before installation so that the Operator can create the required Persistent Volume Claims.

[IMPORTANT]
====
Before proceeding with the installation, verify that a default `StorageClass` is configured in your cluster to ensure that the Quay and Clair components can successfully provision their required persistent volumes.
====
