[id="createorgrobotfederation"]
= createOrgRobotFederation 

[role="_abstract"]
Create a federation configuration for the specified organization robot.

[id="createorgrobotfederation-post"]
== POST /api/v1/organization/{orgname}/robots/{robot_shortname}/federation

Create or replace the federation configuration for the specified organization robot.

**Authorizations: **oauth2_implicit (**user:admin**)

[id="createorgrobotfederation-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"] 
|=== 
|Type|Name|Description|Schema 
|path|*orgname* + *robot_shortname*
_required_|The name of the organization and the short name for the robot, without any user or organization prefix|string 
|===

[id="createorgrobotfederation-request-body-schema"]
== Request body schema (application/json)

Array of federation configuration entries for the robot.

[IMPORTANT]
====
In {productname} 3.18, create and update requests persist `issuer` and `subject` only. The optional `audiences` field is not stored from API requests, and the {productname} v2 UI federation modal does not provide an `audiences` field. When `audiences` is present in stored federation configuration, {productname} validates the token audience during federated robot token exchange. When it is absent, audience validation is skipped and {productname} logs a deprecation warning.
====

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|*issuer*
 
_required_|Issuer URL of the external OIDC provider for this federation entry|string
|*subject*
 
_required_|Subject (`sub` claim) of the external OIDC identity authorized for this robot|string
|*audiences*
 
_optional_|List of acceptable token audience (`aud` claim) values for this federation entry. Reserved for future API support; not persisted by create or update requests in {productname} 3.18.|array of strings
|===

.Example request body
[source,json]
----
[
  {
    "issuer": "https://login.microsoftonline.com/<tenant-id>/v2.0",
    "subject": "<user-object-id>"
  }
]
----
