[id="getrepobuildstatussuperuser"]
= getRepoBuildStatusSuperUser

[role="_abstract"]
Return the status for the builds specified by the build uuids.

[id="getrepobuildstatussuperuser-get"]
== GET /api/v1/superuser/{build_uuid}/status



**Authorizations: **oauth2_implicit (**super:user**)


[id="getrepobuildstatussuperuser-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**build_uuid** + 
_required_|The UUID of the build|string
|===


[id="getrepobuildstatussuperuser-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===
