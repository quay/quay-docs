[id="listusernotifications"]
= listUserNotifications

[role="_abstract"]
List User Notifications.

[id="listusernotifications-get"]
== GET /api/v1/user/namespacenotifications



**Authorizations: **oauth2_implicit (**user:admin**)


[id="listusernotifications-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="listusernotifications-example-command"]
== Example command

[source,terminal]
----
$ curl -X GET \
-H "Authorization: Bearer <bearer_token>" \
-H "Accept: application/json" \
https://<quay-server.example.com>/api/v1/user/namespacenotifications
----
