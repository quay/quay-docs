[id="getorganizationcollaborators"]
= getOrganizationCollaborators

[role="_abstract"]
List outside collaborators of the specified organization.

[id="getorganizationcollaborators-get"]
== GET /api/v1/organization/{orgname}/collaborators



**Authorizations: **oauth2_implicit (**org:admin**)


[id="getorganizationcollaborators-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**orgname**
 
_required_|The name of the organization|string
|===


[id="getorganizationcollaborators-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|200|Successful invocation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===
