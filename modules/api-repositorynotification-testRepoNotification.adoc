[id="testreponotification"]
= testRepoNotification

[role="_abstract"]
Queues a test notification for this repository.

[id="testreponotification-post"]
== POST /api/v1/repository/{repository}/notification/{uuid}/test



**Authorizations: **oauth2_implicit (**repo:admin**)


[id="testreponotification-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**repository** + 
_required_|The full path of the repository. e.g. namespace/name|string
|path|**uuid** + 
_required_|The UUID of the notification|string
|===


[id="testreponotification-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|201|Successful creation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="testreponotification-example-command"]
== Example command
[source,terminal]
----
$ curl -X POST \
  -H "Authorization: Bearer <bearer_token>" \
  https://<quay-server.example.com>/api/v1/repository/<repository>/notification/<uuid>/test
----
