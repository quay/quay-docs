[id="approveservicekey"]
= approveServiceKey

[role="_abstract"]
Approve Service Key.

[id="approveservicekey-post"]
== POST /api/v1/superuser/approvedkeys/{kid}



**Authorizations: **oauth2_implicit (**super:user**)


[id="approveservicekey-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**kid** + 
_required_|The unique identifier for a service key|string
|===


[id="approveservicekey-request-body-schema"]
== Request body schema (application/json)

Information for approving service keys

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**notes** + 
_optional_|Optional approval notes|string
|===


[id="approveservicekey-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema

|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;

|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="approveservicekey-example-command"]
== Example command

[source,terminal]
----
$ curl -X POST \
  -H "Authorization: Bearer <bearer_token>" \
  -H "Content-Type: application/json" \
  -d '{
        "notes": "<approval_notes>"
      }' \
  "https://<quay_server>/api/v1/superuser/approvedkeys/<kid>"
----
