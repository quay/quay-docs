= Configuring a {productname} superuser

[role="_abstract"]
To configure a superuser account for your proof of concept {productname} deployment, you can add the `SUPER_USERS` array to `config.yaml`. Superusers can manage users, organizations, service keys, and global settings from the admin panel.

.Procedure

. Add the `SUPER_USERS` array to the `config.yaml` file:
+
[source,yaml]
----
SERVER_HOSTNAME: quay-server.example.com
SETUP_COMPLETE: true
SUPER_USERS:
  - quayadmin
...
----
