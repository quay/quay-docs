[id="createorganizationquota"]
= createOrganizationQuota

[role="_abstract"]
Create a new organization quota.

[id="createorganizationquota-post"]
== POST /api/v1/organization/{orgname}/quota



**Authorizations: **oauth2_implicit (**super:user**)


[id="createorganizationquota-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**orgname**
 
_required_||string
|===


[id="createorganizationquota-request-body-schema"]
== Request body schema (application/json)

Description of a new organization quota

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**limit_bytes**
 
_required_|Number of bytes the organization is allowed|integer

|**limits**
 
_optional_|Human readable storage capacity of the organization. Accepts SI units like Mi, Gi, or Ti, as well as non-standard units like GB or MB. Must be mutually exclusive with `limit_bytes`.|string
|===


[id="createorganizationquota-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|201|Successful creation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===


[id="createorganizationquota-example-command"]
== Example command

[source,terminal]
----
$ curl -X POST "https://<quay-server.example.com>/api/v1/organization/<orgname>/quota" \
     -H "Authorization: Bearer <access_token>" \
     -H "Content-Type: application/json" \
     -d '{
         "limit_bytes": 10737418240,
         "limits": "10 Gi"
     }'
----
