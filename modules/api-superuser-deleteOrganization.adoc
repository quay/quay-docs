[id="deleteorganization"]
= deleteOrganization

[role="_abstract"]
Deletes the specified organization.

[id="deleteorganization-delete"]
== DELETE /api/v1/superuser/organizations/{name}



**Authorizations: **oauth2_implicit (**super:user**)


[id="deleteorganization-path-parameters"]
== Path parameters

[options="header", width=100%, cols=".^2a,.^3a,.^9a,.^4a"]
|===
|Type|Name|Description|Schema
|path|**name**
 
_required_|The name of the organizaton being managed|string
|===


[id="deleteorganization-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|204|Deleted|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="deleteorganization-example-command"]
== Example command

[source,terminal]
----
$ curl -X DELETE \
  -H "Authorization: Bearer <bearer_token>" \
  "https://<quay_server>/api/v1/superuser/organizations/<organization_name>"
----
