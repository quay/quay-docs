[id="createapptoken"]
= createAppToken

[role="_abstract"]
Create a new app specific token for user.

[id="createapptoken-post"]
== POST /api/v1/user/apptoken



**Authorizations: **oauth2_implicit (**user:admin**)



[id="createapptoken-request-body-schema"]
== Request body schema (application/json)

Description of a new token.

[options="header", width=100%, cols=".^3a,.^9a,.^4a"]
|===
|Name|Description|Schema
|**title**
 
_required_|Friendly name to help identify the token|string
|===


[id="createapptoken-responses"]
== Responses

[options="header", width=100%, cols=".^2a,.^14a,.^4a"]
|===
|HTTP Code|Description|Schema
|201|Successful creation|
|400|Bad Request|&lt;&lt;_apierror,ApiError&gt;&gt;
|401|Session required|&lt;&lt;_apierror,ApiError&gt;&gt;
|403|Unauthorized access|&lt;&lt;_apierror,ApiError&gt;&gt;
|404|Not found|&lt;&lt;_apierror,ApiError&gt;&gt;
|===

[id="createapptoken-example-command"]
== Example command
[source,terminal]
----
 $ curl -X POST \
  -H "Authorization: Bearer <access_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "MyAppToken"
  }' \
  "http://quay-server.example.com/api/v1/user/apptoken"
----
