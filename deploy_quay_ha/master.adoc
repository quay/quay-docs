= Deploy {productname} - High Availability

[role="_abstract"]
To deploy a high availability, enterprise-quality {productname} registry, you can configure load balancing, a PostgreSQL database, Ceph object storage, Redis, and multiple registry nodes.

[IMPORTANT]
====
The {productname} configuration tool is unsupported for 
standalone high availability deployments. Create and maintain 
the `config.yaml` file manually, as described in this guide.
====
