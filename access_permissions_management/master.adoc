= Managing access and permissions
:context: quay-security

[role="_abstract"]
{productname} provides a comprehensive permissions model so that you can control who can access, manage, and modify repositories. You can manage user access, team roles, and repository visibility by using the UI or the API.

You can complete tasks such as the following:

* Configure role-based access controls
* Adjust repository visibility
* Create and manage robot accounts
* Review Clair vulnerability reporting
